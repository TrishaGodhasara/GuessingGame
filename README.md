# 🧩 Guess The Company

> **A fun brain teaser for curious minds!**
>
> Take a quick break from work, coding, or studying and challenge yourself to identify famous brands using progressively revealing hints. Whether you're into technology, fashion, automobiles, food, sports, or lifestyle brands, this game is a fun way to test your memory while discovering more about the companies around us.

---

## 🎯 About the Project

**Guess The Company** is an interactive full-stack web application where players identify well-known brands using a series of carefully designed hints.

The fewer hints you use, the higher your score!

This project combines entertainment with learning by helping users recognize and learn about brands across different industries.

---

## ✨ Features

- 🧠 Brain teaser designed for quick breaks
- 💡 Progressive hint system
- 🎯 Score based on number of hints used
- 👤 Username-based game sessions
- 🔄 Randomized brand selection
- 📊 Score tracking throughout the session
- 🚫 Already completed brands won't repeat
- 📱 Responsive and modern user interface
- ⚡ Fast and interactive gameplay

## 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/TrishaGodhasara/GuessingGame.git
```

### 2️⃣ Navigate to the project

```bash
cd GuessingGame
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Run the application

```bash
npm run dev
```

This starts both the frontend and backend simultaneously.

---

## 🎮 How to Play

1. Enter your username.
2. A random company is selected.
3. Read the first hint.
4. Guess the company.
5. If you're unsure, reveal more hints.
6. Score more points by guessing with fewer hints.
7. Continue until all companies are completed.

---

## 🎯 Scoring

- ✅ Correct guesses earn points.
- 💡 Using fewer hints results in a higher score.
- 🏆 Challenge yourself to achieve the highest score possible.
